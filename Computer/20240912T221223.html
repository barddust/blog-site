前言

迫于生活压力，终于还是从 Linux 换回 Windows 了，说实话，真的太逊了，日常 CPU 风扇乱转， Emacs 配置和编码重新大改，平铺窗口没法用 Win 键，除了玩游戏方便，其他真没什么优势。基于这种环境下，尝试去配置一个日用舒适的 Windows，本篇用以记录。

后记

你问我好用吗，我的回答是一坨。用起来各方面都不如 Linux，包括速度、习惯等等。目前 Windows 只是工作使用，日用还得是 Linux ； 尤其是 Emacs, 在 Windows 下的体验极差，很多功能用不上， VS Code 用得也不舒服。
