导数的运算

Question 1.1
  1. 已知函数 \(f(x) = \frac{x^2}{2} + 2x - 3\ln x\) ，求$f(x)$的导数，并求出 \(f'(x) > 0\) 的解集.

Solution

\(f'(x) = x + 2 - \frac{3}{x} = \frac{x^2+2x-3}{x}\). 要使 \(f'(x) > 0\) 有：

Case I, $ x > 0 \land x^2 + 2x - 3 > 0$, then \(x>1\).

Case II, $ x < 0 \land x^2 + 2x - 3 < 0$, then \(-3 < x < 0\).

综上，解集为 \((-3,0)\cup(3,+\infty)\). □
